# 🌦️ CosechaClima — Tu semáforo de decisiones climáticas para el campo

Un productor de Carazo tiene minutos, no horas, para decidir qué hacer antes de que llegue una tormenta. **CosechaClima** convierte la incertidumbre climática en **3 acciones claras y priorizadas**, mediante un sistema de semáforo (🟢 verde / 🟡 amarillo / 🔴 rojo) diseñado para pequeños productores de maíz y frijol.

Ante cada evento meteorológico relevante, la app cruza el riesgo climático con la etapa fenológica del cultivo del usuario y responde una sola pregunta: **¿qué hago ahora mismo para salvar mi cosecha?**

Sin jerga técnica. Sin reportes de 20 páginas. Solo lo que importa, cuando importa.
