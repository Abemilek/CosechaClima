# 📱 CosechaClima — App Móvil (Flutter)

## Cómo empezar

Desde la raíz del monorepo:

```bash
cd mobile
flutter create . --org com.curcarazo --project-name cosecha_clima
```

## Conexión con el backend

La API vive en `../backend` (ASP.NET Core). Ver [`../backend/README.md`](../backend/README.md) para la lista de endpoints disponibles (`/api/motor/semaforo`, `/api/usuarios/registro`, etc.).
